# TriCraft Handyman Services

Professional handyman website for TriCraft Handyman Services LLC — serving Ephrata, PA and all of Lancaster County with plumbing, electrical, and carpentry services.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Update Content

Edit `/lib/constants.ts` — all company info, services, testimonials, and contact details live in one file.

## Deploy

Push to `main` branch → auto-deploys on Vercel.
